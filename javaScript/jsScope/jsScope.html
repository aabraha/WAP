  2. What is the difference between a method and function?

    A function is a group of statements dedicated to perform a certain task. A function in JavaScript is an object. A method, on the other hand, is a function that belongs to a javascript object.

  3. What does 'this' refer to when used in a Java method?

    this keword is used in Java to refer to the object that called the method.

  4. What does 'this' refer to when used in a JavaScript method?

    this keyword is used in JavaScript to refer to the object calling the method(the owner object).

  5. What does 'this' refer to when used in a JavaScript constructor function?

    this keyword is used in JavaScript to refer to the owner object.

  6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to?

    this keyword is used in JavaScript to refer to the owner object in this case Object x.

  7. What is a free variable in JavaScript?

    A free variable is a variable used within a function, which is neither a formal parameter to the function nor defined in the function's body (and in scope at the point of the variable's use).
